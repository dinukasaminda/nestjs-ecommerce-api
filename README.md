# Ecommerce application

## Install dependencies

```bash
yarn install
```

## Run database server on docker

```bash
docker-compose up -d
```

## Run application on local

```bash
yarn start:dev
```

## Swagger UI

http://localhost:3000/swagger-ui

## Swagger JSON

./swagger.json

## Open Swagger Editor

```bash
go to :
https://editor-next.swagger.io/
copy and paste swagger.json content


```
